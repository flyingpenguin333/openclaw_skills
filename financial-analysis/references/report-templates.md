# Report Templates

## Equity Research Report Structure

### 1. Executive Summary
- Investment thesis (1-2 paragraphs)
- Key metrics table
- Price target and rating

### 2. Company Overview
- Business description
- Industry overview
- Competitive positioning

### 3. Investment Thesis
- Bull case (3-5 key points)
- Bear case (risks)
- Key catalysts

### 4. Financial Analysis
- Historical performance
- Projections (3-5 years)
- Key metrics (margins, growth, returns)

### 5. Valuation
- Comps summary
- DCF assumptions and output
- Sensitivity analysis
- Price target derivation

### 6. Risks
- Downside scenarios
- Key uncertainties

## Pitch Book Structure

### Section 1: Market Overview
- Industry trends
- Market size and growth
- Competitive landscape

### Section 2: Company Profile
- Business overview
- Financial highlights
- Growth strategy

### Section 3: Valuation
- Trading comps
- Transaction comps
- DCF analysis
- Football field

### Section 4: Recommendation
- Strategic alternatives
- Timeline
- Next steps

## One-Pager Format

Single page with:
- Company logo/header
- Investment thesis (2-3 bullets)
- Key financials table
- Valuation summary
- Price target
- Risk factors (brief)

## IC Memo Structure

### 1. Deal Summary
- Target company
- Transaction type
- Investment amount
- Expected returns

### 2. Business Overview
- Industry
- Revenue model
- Growth drivers

### 3. Financial Analysis
- Historical performance
- Projections
- Key assumptions

### 4. Returns Analysis
- MOIC
- IRR
- Exit scenarios

### 5. Risks and Mitigants
- Key risks
- How to mitigate

### 6. Recommendation
- Proceed/Decline
- Key conditions

## Formatting Tips

- Use consistent fonts (Arial/Calibri for body, larger for headers)
- Number pages
- Include date and author
- Source all data
- Use charts for trends
- Keep it concise - less is more
